Курсова задача #7
"Показване на Python-програма"

Да се напише програма, която визуализира кода на програмен файл, написан на
езика Python по подобие на начина, който се използва в "Notepad++" (виж
изображението след заданието).
• В текста да се оцветяват имена на оператори, служебните символи, имена
на типове, директивите за предпроцесора, символните низове и
коментарите.
За навигация по текста да се използват следните клавиши:
• Home, End - за позициониране в началото и в края на текста.
• PgUp, PgDn - за преместване на една страница напред и назад по текста
· left, up, right, down - за навигация по текста
· CTRL+'<' , CTRL+'>' - за позициониране в началото и в края на текущия.
Визуализацията на курсора е по избор на разработчика.
Текстовете на кирилица и латиница трябва да бъдат коректно изписани.
Програмата трябва да работи в среда "Linux".

# Python program to display all the prime numbers within an interval

lower = 900
upper = 1000

print("Prime numbers between", lower, "and", upper, "are:")

for num in range (lower, upper + 1):
# all prime numbers are greater than 1
    if num > 1:
        for i in range(2, num):
            if (num % i) == 0:
                break
        else:
            print(num)
